       disabled: false

   In settings.yml, you need:

     formats:
       - html
       - json    ← This line must exist!

3️⃣  DISABLE RATE LIMITING
   
   In settings.yml:
   server:
     limiter: false

4️⃣  RESTART SEARXNG
   
   docker restart searxng
   
   Wait 30 seconds, then test again.
